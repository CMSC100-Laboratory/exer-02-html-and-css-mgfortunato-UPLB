Steps on how to create GitHub page:
	1. Go to github
	2. Click Settings
	3. Scroll down in general (default) until you see the "Danger Zone"
	4. Click Change Repository Visibility
	5. Make it public
	6. Scroll up
	7. On the left side, click the "Pages" tab
	8. In branch, choose what branch you want your website to use
	9. Click save then wait for GitHub to load your website
	10. After loading, you will see "Your site is live at <domain>".
	11. Enjoy your web creation!

Key takeaways:
	- GitHub is a great tool for storing the progress of our programs
	- It is also a great way for hosting websites
	- There is an endless possibility in designing using CSS, there is so much to learn but there is always the internet to help us learn about CSS more.
